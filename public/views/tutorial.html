Projects

You can create your own projects by clicking the "new project" button. All changes you do in a project will be saved automatically. Projects can be deleted by clicking "x" on the project you want to delete.

Layers

Layers contains different spatial data, example rivers, schools and so on.

You can view and hide layers by clicking the , and you can delete a layer by clicking . You can also change the order the layers are drawn by dragging the layer to a different place in the layer menu. The layers at the top of the menu will be positioned at the front, on top of the other layers.

You can also change styling of a layer by clicking the icon, and you can then change color, opacity and line width.

Buffer

A buffer is an area defined as a zone around a map layer. The zone is measured by a distance of meter from the points in the input layer.

Input: Polygon, Lines, Points
Output: Polygon

Intersection

Intersection is defined as the overlapping area between two polygons.

Input: Polygon
Output: Polygon

Difference

Difference takes two parameters; Output area and intersecting area. The function returns all area that are in the "output area" but not in the "intersecting area".

Input: Polygons
Output: Polygon

Merge

Merge takes two layers of same type and merge them together into one layer.

Input 1: Polygon, Point, Line
Input 2: Same type as selected in "Input 1"
Output: Same type as selected in "Input 1" and "Input 2"

Within

Within returns all elements in a layer that is completely or partially inside another selected layer.

Input 1: Polygon, Point
Input 2: Polygon
Output: Same type as selected in "Input 1"

Upload layers

You can upload your own data to your project, by clicking "Add new layer". A good place to find data is on overpass-turbo which you can find a link to in the upload layer popup window. If you upload a layer that contains multiple types of data, the layer will be split into different layers that contains each of the types.

Input: GeoJSON
